# Todo-List CLI 📝

Um sistema gerenciador de tarefas em linha de comando (CLI) desenvolvido em JavaScript para consolidação de fundamentos de lógica de programação, manipulação de arrays, objetos e funções estruturadas no ambiente Node.js.

## 🚀 Funcionalidades

- **Adicionar Tarefa:** Registo de tarefas com ID automático e estado inicial pendente.
- **Listar Tarefas:** Visualização organizada de todas as tarefas e os seus respetivos estados (concluída/pendente).
- **Concluir Tarefa:** Alteração do estado de uma tarefa específica através do seu ID.
- **Remover Tarefa:** Exclusão de tarefas do sistema.

## 🛠️ Tecnologias Utilizadas

- **JavaScript (ES6+)**
- **Node.js** (Ambiente de execução)
- **readline-sync** (Biblioteca para entrada de dados no terminal)
